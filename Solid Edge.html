Description

Solid edge is the full featured 2D drafting and 3D modelling software. The Solid Edge is not only the 3D parametric feature-based modelling system. It supports the new solid modelling technology named synchronous modelling. This technology combines the direct modelling with the dimension driven design. Parametric relationships can be applied directly to the solid features without having to depend on 2D sketch geometry, and common parametric relationships are applied automatically. The usage of the synchronous technology is beneficial I the topic of simple part modelling or 3D scanned part reconstruction, further construction, or repair. The cons of the Synchronous modelling to harder to modify the modelling steps than the traditional feature-based modelling.

The 3D modelling space gives an opportunity to reduce the engineering costs: it can be reusing the 2D drafts to modelling parts for example dxf drawing. The main importance for the modelling in 3D space to make the interference checking and the fit and position checking before the manufacturing. With the help of the program, we can build the entire 3D digital prototypes and can optimize the design before the production process. It can be an opportunity to design casted, machined or stylized components and define lever-age process specific frames, piping, tubes wiring and weldment design too. The digital twin (which can use with other application) or the animation, rendering and exploded views using helps to analyze the construction in a digital space and achieve higher level of products quality in less time.

The sheet metal design module gives a large-scale usage in a manufacturing design documentation. With the help of this module the user can create flanges, holes, relief, and corners with different options. The main definition is similar to the part definition except two more additional group of features:

  • Material definition (density, thickness, bend, radius, relief depth and width
  • Flat pattern definition

When sheet metal is formed into a bend, the metal on the inside of the bend is compressed and the material on the outside of the bend is stretched around an imaginary line along the thickness of the material sometimes call the neutral line or as Solid Edge refers to it as the PZL= Plastic Zone Length. The user defined or the built-in bend equation tab defines the formula which we can use for the calculation. The 2D technical drawing consist the bend tables which represents the sheet metal feature type, the radius, the angle and the direction too. On the other case the program gives an opportunity to define the sheet metal parts with triangulation lines (this important in a case of example conical shapes).

The 2D drafting system is reliable when use standard manufacturing systems. The drafting module create wide variety of views (auxiliary, standard, section, detail, broken and isometric). The drafting module helps to define standard (built-in) or customized part list table and balloons too. The drawing views have 2 way communication with the model and keep up to date the documentation system. The industry standard symbols have been built into the software.

The program able to import or export parts of assemblies to universal exchange formats, example: step, iges, Parasolid, dxf, dwg, Acis, NX.

The solid edge Shining 3D edition includes the full reverse engineering platform with generative design and FEA simulation too.

Biotec Srl in the dental manufacturing side Solid edge and Simcenter Femap have been used from 2010. With the help of the Siemens programs the solid and surface 3D models easy converted into a machining program which reduced the number of the wrong or faulty programs and the machining time.

Features

SolidWorks software enables the user to:

  • Create parts, assemblies, and 2D drawings
  • Use specialized tools for sheet metal, routing
  • Design with cost estimation, manufacturability and functional assembly
  • Ensure successful design with simulation
  • Build toolpaths with integrated paths
  • Conveniently reuse existing CAD and common parts
  • Create photorealistic renderings
  • Reverse engineering for scan to solid geometry
