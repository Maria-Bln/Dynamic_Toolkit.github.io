Description

Autodesk Maya (or commonly shortened Maya) is a 3D digital tools to modelling and make animation in a digital space. Maya incorporates the natural laws of physics to control the behavior of virtual objects in computer animation. It can be used for movies, games, and commercials. With the help of the program the user defines heavy models, which help to give ultra 3D effects, realistic view.

Maya use surface modelling system. To define the surfaces Maya use 3 different methods:

  • Polygons
  • NURBS (Non-Uniform Rational Basic Splines)
  • SubD’s

The polygon models are made of vertices, edges, and faces. With these primitives the program defines polygon primitives like cylinder, sphere, or cube. The pulling, pushing, adding relation between these forms define the final shape of the model. The polygon modelling is the simplest modelling strategy, but the final forms have sharp edges, or we need to define large number of primitives to the realistic form. The NURBS surface definition gives an opportunity to define soft edges and form with the help of few points. The NURBS curve have been defined with mathematical expression. The SubD’s have characteristics of both polygons and NURBS surfaces; they are made up of vertices, faces & edges like polygons, but use curves like NURBS.

In the engineering life the motion definition and the physical properties definition is the importance of the Maya. The program is universal, not requirement mechanical engineer studies to make an assembly or a movie. Typically, the movie based modelling is current with Maya.

When we want to make a model is maya we need to define the position and the size of primitives in the model space. After that with the selection tool we can modify the feature properties. Textures can be assigned to the solids and the motion path have been defined to make an animation.

With the help of this software full detailed movie has been created. With the polygonal, NURBS or SubD’s surface modelling system detailed parts have been defined and the texture representation helps to make a similarity of the solid with the real life solid. The surface modelling system gives an opportunity to define very complex surface definitions. For more editing or example 3D printing Maya supports the IGES, STEP, STL and some other file formats

Summary

Adobe Maya software enables the user to:

  • Create models with surface modelling tools
  • Create Polynomial, NURBS, or SubD’s surfaces
  • Define textures for the surfaces
  • Make a render for the scene
  • Define a motion and make a movie
